# v0.9.392 - Sistema de Diagnóstico Inteligente de Herramientas MCP

## Resumen
Nueva herramienta MCP `diagnose_tool_health` que analiza errores históricos de ejecuciones de herramientas MCP y genera reportes priorizados para mejorar la confiabilidad del sistema.

## Cambios Realizados

### Nueva Herramienta: `mcp_omnysystem_diagnose_tool_health`
- **Archivo**: `src/layer-c-memory/mcp/tools/diagnose-tool-health.js`
- **Descripción**: Analiza la salud de las herramientas MCP ejecutadas
- **Funcionalidades**:
  - Identifica herramientas con alta tasa de fallo
  - Detecta patrones de errores frecuentes
  - Calcula métricas de rendimiento (duración promedio, tasa de éxito)
  - Genera diagnóstico priorizado con recomendaciones
  - Soporte para filtrar por herramienta específica
  - Modo resumen vs. detalles completos

### Métricas Analizadas
1. **Tasa de Éxito**: Porcentaje de ejecuciones exitosas por herramienta
2. **Duración Promedio**: Tiempo promedio de ejecución en milisegundos
3. **Errores Frecuentes**: Patrones de errores que se repiten
4. **Prioridad de Issues**: Clasificación por severidad (alta/media/baja)

### Reporte Generado
El diagnóstico incluye:
- Resumen general del sistema de herramientas
- Lista de herramientas con sus métricas de salud
- Issues priorizados por severidad
- Patrones de errores detectados
- Recomendaciones de mejora

### Integración con Sistema de Métricas
- Utiliza datos de `mcp_tool_runs` para análisis histórico
- Complementa `get_recent_errors` con análisis más profundo
- Proporciona contexto para mejorar el `repair_rate`

## Casos de Uso

### 1. Identificar Herramientas Problemáticas
```javascript
// Analizar todas las herramientas
diagnose_tool_health({ limit: 100 })

// Resultado: Lista de herramientas con tasa de fallo > 20%
```

### 2. Diagnosticar Herramienta Específica
```javascript
// Analizar solo split_large_file
diagnose_tool_health({ 
  toolName: 'mcp_omnysystem_split_large_file',
  limit: 50 
})
```

### 3. Monitoreo Rápido
```javascript
// Modo resumen para revisión rápida
diagnose_tool_health({ 
  limit: 100,
  includeDetails: false 
})
```

## Impacto Esperado
- **Mejora en Repair Rate**: Identificar y corregir errores recurrentes
- **Reducción de Thrashing**: Priorizar fixes de herramientas con más fallos
- **Monitoreo Proactivo**: Detectar problemas antes que escalen
- **Documentación Automática**: Generar reportes de salud sin esfuerzo manual
