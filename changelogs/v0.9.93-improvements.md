# Mejoras a Atomic Edit - Sistema de Ayuda Inteligente

## Problema Original

Cuando `atomic_edit` fallaba porque `oldString` no se encontraba exacto, el sistema decía:

```json
{
  "error": "TRANSACTION_ROLLED_BACK",
  "message": "Validation failed: oldString not found: async buildSocieties() { logger.info('...",
  "success": false
}
```

**¡Esto era inútil!** No ayudaba al usuario a entender:
- ¿Cuál es el contenido REAL del archivo?
- ¿Por qué no matcheó? (whitespace, encoding, etc.)
- ¿Qué debería usar en su lugar?

---

## Solución Implementada

### 1. ModifyOperation - Sugerencias Inteligentes

**Archivo:** `src/core/atomic-editor/operations/modify-operation.js`

Ahora cuando `oldString` no se encuentra, el sistema:

1. **Busca similitudes** usando 3 estrategias:
   - `WHITESPACE_MISMATCH`: Mismo contenido pero diferente indentación
   - `CONTEXT_MATCH`: Encontró la primera línea, sugiere el contexto completo
   - `PARTIAL_MATCH`: El contenido empieza igual pero difiere después

2. **Muestra preview del archivo** con las líneas relevantes

3. **Da recomendaciones** específicas de cómo arreglarlo

### 2. Ejemplo de Respuesta Nueva

```json
{
  "error": "MUTATION_FAILED",
  "message": "oldString not found",
  "success": false,
  "severity": "critical",
  "help": {
    "whatYouProvided": "    async buildSocieties() {\n        logger.info('[SocietyEngine] Building societies...');",
    "recommendation": "Use one of the suggested strings above, or read the file first to see the exact content",
    "suggestions": [
      {
        "type": "WHITESPACE_MISMATCH",
        "confidence": "high",
        "line": 23,
        "suggestedString": "async buildSocieties() {\n        logger.info('[SocietyEngine] Building societies...');",
        "reason": "Same content but different indentation/whitespace"
      },
      {
        "type": "CONTEXT_MATCH", 
        "confidence": "medium",
        "line": 23,
        "suggestedString": "async buildSocieties() {\n        logger.info('[SocietyEngine] Building societies...');\n        \n        const startTime = Date.now();",
        "reason": "Found 'async buildSocieties()' at line 23"
      }
    ]
  },
  "filePreview": "23: async buildSocieties() {\n24:     logger.info('[SocietyEngine] Building societies...');\n25:     \n26:     const startTime = Date.now();\n...",
  "recommendation": "Use one of the suggested strings above, or read the file first to see the exact content"
}
```

## Archivos Modificados

1. **`src/core/atomic-editor/operations/modify-operation.js`**
   - Agregado método `_generateSuggestions()` - busca strings similares
   - Agregado método `_getFilePreview()` - extrae contexto relevante
   - Mejorado mensaje de error para incluir `help`, `suggestions`, `filePreview`

2. **`src/layer-c-memory/mcp/tools/atomic-edit/atomic-editor-tool.js`**
   - Propaga información de ayuda desde el atomic editor

3. **`src/layer-c-memory/mcp/core/shared/base-tools/atomic-mutation-tool.js`**
   - Incluye información de ayuda en errores de transacción

## Cómo usar las sugerencias

Cuando `atomic_edit` falle, la respuesta incluirá:

```javascript
// Si hay sugerencias de WHITESPACE_MISMATCH
if (response.suggestions?.[0]?.type === 'WHITESPACE_MISMATCH') {
  // Usar exactamente el string sugerido (con el whitespace correcto)
  const correctedString = response.suggestions[0].suggestedString;
  // Reintentar con correctedString
}

// Si no hay sugerencias, leer el archivo primero
if (!response.suggestions || response.suggestions.length === 0) {
  // Usar query_graph o readFile para ver el contenido exacto
}
```
