# Changelog OmnySys v0.9.79

## 🧬 Refactorización Genética (Consolidación SQL)
- **BaseSqlRepository**: Introducción de una clase base abstracta para centralizar operaciones SQLite, garantizando consistencia y reduciendo código boilerplate.
- **Consolidación de Handlers**: Refactorización de `SemanticHandler`, `DependencyHandler` y `RiskHandler` para utilizar la nueva infraestructura común.
- **Remoción de Duplicados**: Eliminación de sentencias SQL `DELETE` y `UPSERT` redundantes en múltiples módulos de caché y almacenamiento.

## 🛡️ Estabilidad del Sistema y Bug Fixes
- **Resolución de Extraction Crash**: Se ha solucionado un error crítico (`Cannot read properties of undefined (reading 'name')`) en el extractor de átomos causado por referencias huérfanas a la base de datos tras la refactorización.
- **Auditoría de Referencias**: Corrección de llamadas obsoletas a `getDb` y `_getDb` en `ram-cache.js` y `shadow-storage.js`.

## 🌡️ Software Physics & DNA
- **Métricas de Física**: Integración de scores de **Gravedad Estructural** y **Reactividad** en el modelo de datos de átomos.
- **Análisis de ADN**: Mejora en la detección de duplicados estructurales para facilitar la purificación automática del código.
- **DNA Purification**: El sistema ahora es capaz de identificar y reportar colisiones genéticas con mayor precisión, permitiendo una reducción drástica de la entropía del sistema.
