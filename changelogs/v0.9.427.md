# OmnySys v0.9.427

## Folderized Name Normalization

- Added a safe folderized-name normalizer that derives its plan from the canonical folderization naming report.
- The normalizer now supports `analyze`, `plan`, and `execute` modes, with execution delegated to the existing rename flow only when the plan is explicitly safe.
- Added propagation contract metadata to the folderized rename/normalization tool definitions so the watcher and contract surfaces can treat naming debt as a canonical compiler contract.
