# v0.9.431 - Sprint 16: MCP Restart Recovery, Observability Contract, and Dashboard Split (2026-04-08)

## Key Achievements

1. Hardened the MCP restart/reconnect path so `restart_server({ processRestart: true })` now returns a valid ACK, recovers the bridge automatically, and survives the reconnect pattern without `Transport closed`.

3. Split `src/shared/compiler/dashboard.js` into a thin coordinator plus `dashboard-sections.js` to push the watcher back under the file-size threshold and avoid future AI-edit debt.
4. Documented the restart bug and added regression coverage for the recovery path, observability contract, and dashboard health wiring.
