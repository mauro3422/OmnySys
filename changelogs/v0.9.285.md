### `src/layer-c-memory/query/queries/file-query/enriched/with-atoms.js`
- **Fix:** Guard clause en `calculateStats()` para manejar caso cuando `derivedMetadata` es null/undefined

### `src/layer-c-memory/storage/database/schema.sql`
- **Improvement:** DEFAULT values agregados a columnas NOT NULL:
  - `line_start`, `line_end`, `lines_of_code` → DEFAULT 0
  - `extracted_at`, `updated_at`, `created_at`, `assessed_at`, `detected_at` → DEFAULT (datetime('now'))
- **Beneficio:** Evita errores de constraint al insertar registros sin todos los campos obligatorios

### `vitest.config.js`
- **Improvement:** Agregados aliases de path `/src` y `/tests` para imports más limpios en tests
