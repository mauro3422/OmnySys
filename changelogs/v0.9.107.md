# Changelog v0.9.107 - Stabilization & Unified Diagnostics

## [0.9.107] - 2026-03-07

### Added
- **Unified Diagnostics Dashboard**: A new high-fidelity terminal summary that consolidates Pipeline Integrity, Technical Debt, and Project Physics.
- **Extended Metrics**: Added reporting for Conceptual Duplicates, Pipeline Orphans, and Atom Type distribution to the dashboard.
- **Reporting Throttle**: Implemented state-based de-duplication in `dashboard-reporter.js` to prevent redundant console output.

### Fixed
- **Windows File Locking**: Resolved a race condition where the worker couldn't delete the database during `--reanalyze` restarts. Cleanup is now handled safely by the Proxy.
- **Semantic Guard Crash**: Fixed a bug where `stripStrings` was used without being imported in several conformance guards, preventing analysis failures.
- **Init Fragmentation**: Suppressed redundant "Phase 2" background logs to ensure the consolidated "FINAL SYSTEM SUMMARY" is the clean concluding output.

### Changed
- Refactored `dashboard-reporter.js` to use project subpath aliases (`#core`, `#layer-c`) and improved internal modularity.
- Moved all restart-related database cleanup from the worker to `mcp-http-proxy.js`.
