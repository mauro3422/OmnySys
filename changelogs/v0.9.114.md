# v0.9.114

## Summary

This release tightens compiler governance conformance so runtime metrics stop reporting false-positive wrapper and async-policy drift after hot reloads and full MCP restarts.

## What Changed

- Tightened canonical extension conformance matching in `src/shared/compiler/canonical-extension-conformance.js`.
  - Shared compiler barrel imports are now parsed from the actual import block only.
  - Wrapper detection now only flags direct passthrough wrappers instead of any function that happens to return a canonical helper plus extra logic.
- Tightened async error conformance restart/timeout detection in `src/shared/compiler/async-error-conformance.js`.
  - Telemetry field names like `restartLifecycle` and `runtimeRestartMode` no longer trigger restart-pressure findings.
  - Detection now focuses on operational timeout/restart signals such as worker restart, retry, reconnect, timeout scheduling, and daemon recovery paths.
- Removed a trivial local wrapper in `src/layer-c-memory/mcp/core/analysis-checker/file-scanner.js`.
  - `hasExistingAnalysis` is now a canonical re-export of `hasPersistedCompilerAnalysis`.

## Verified Outcome

- `get_server_status().compilerExplainability.policySummary.total = 0`
- `get_server_status().compilerExplainability.standardization.adoptionGaps = []`
- `get_server_status().compilerExplainability.compilerContractLayer.summary.canonicalWrapperFindings = 0`
- `get_server_status().compilerExplainability.compilerContractLayer.summary.parallelCanonicalSurfaceFindings = 0`
- `aggregate_metrics(pipeline_health).healthScore = 96`
- `aggregate_metrics(pipeline_health).grade = A`
- Remaining compiler warning is now only the expected semantic advisory:
  - `semantic_connections` is healthy as an advisory surface, while `atom_relations` remains the source of truth.

## Notes

- This release does not change the canonical semantic contract itself; it removes governance false positives so the existing contract layer is easier to trust.
- Local deletion of `.omnysys.db` was left untouched and is not part of this release.
