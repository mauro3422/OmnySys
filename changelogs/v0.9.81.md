# Changelog OmnySys v0.9.81

## 🛡️ Escudo de Análisis (Extractor Shield)
- **Defensa Multinivel**: Se han implementado filtros defensivos en `extractor.js` y `typescript.js` para ignorar elementos indefinidos en exportaciones y usos de tipos. Esto resuelve de raíz el crash `Cannot read properties of undefined (reading 'name')` que afectaba a archivos con estructuras de código atípicas o scripts de depuración.
- **Robustez del Orquestador**: El motor de análisis ahora es resiliente a fallos menores en los extractores individuales, garantizando que el indexado nunca se detenga.

## ⚛️ Gravedad Confirmada
- **Métricas de Física**: Verificación exitosa de la persistencia de `importance_score` y `coupling_score` en la base de datos de átomos.
- **DNA Purification**: El sistema opera con una base genética limpia y sin duplicados SQL.
