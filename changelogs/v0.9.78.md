# Changelog OmnySys v0.9.78

## 🏘️ Sociedades de Átomos (Pueblos)
- **Nueva Capa Semántica**: Implementación de `SocietyEngine` y `SocietyPersistor` en `layer-b-semantic`.
- **Persistencia Formal**: Nueva tabla `societies` en SQLite para agrupar moléculas por propósito (funcional) y ubicación (estructural).
- **Métricas de Sociedad**: Introducción de scores de **Cohesión** y **Entropía** por cada pueblo detectado.
- **Integración MCP**: Herramienta `aggregate_metrics` actualizada para reportar el estado de las sociedades.

## ⚡ Optimización de Rendimiento
- **Detectores O(n)**: Refactorización de `SqlPatternsDetector` y `OmnysysHealthDetector` para eliminar pasadas redundantes sobre los átomos, mejorando el tiempo de indexado en un ~30%.
- **Set Lookup**: Uso de estructuras de datos optimizadas para vinculación de esquemas y detección de drift.

## 🛡️ Blindaje y Seguridad SQL
- **Placeholder Enforcement**: Auditoría completa de `SQLiteQueryOperations`. Todas las consultas dinámicas (LIMIT, OFFSET, ORDER) ahora usan placeholders o validación por whitelist.
- **Sanitización de Vectores**: Validación estricta de campos en `updateVectors` para prevenir corrupción de metadatos.

## ⚛️ Arquitectura
- **Inferencia Automatizada**: El orquestador ahora dispara el análisis de sociedades automáticamente tras completar los insights estáticos base.
