# Changelog v0.9.393 - Tool Health Trending & Automatic Alerts

> **Fecha:** 2026-04-02  
> **Tipo:** Feature enhancement - Diagnóstico proactivo

## 🆕 Nuevas Features

### Tool Health Trending System
- **Nuevo servicio compartido:** `src/shared/compiler/tool-health-trending.js`
  - `calculateToolTrend()` - Calcula trending de métricas comparando mitades temporales
  - `generateAutomaticAlerts()` - Genera alertas automáticas basadas en umbrales configurables
  - `getDailyToolMetrics()` - Obtiene métricas diarias de `mcp_tool_runs`
  - `formatToolHealthDashboard()` - Dashboard visual ASCII para terminal

### Umbrales de Alertas Automáticas
```js
ALERT_THRESHOLDS = {
  successRate: { critical: 50%, warning: 80%, healthy: 95% },
  repairScore: { critical: <-3, warning: <-1, healthy: >=0 },
  avgDuration: { slow: >5s, critical: >10s },
  thrashingCount: { warning: >=3, critical: >=5 }
}
```

## 📊 diagnose_tool_health v2.0

### Nuevos features
- **Trending de métricas** - Compara primera vs segunda mitad del período
- **Alertas automáticas** - Se disparan cuando métricas cruzan umbrales
- **Dashboard visual ASCII** - Formato de tabla para terminal
- **Métricas diarias** - Para análisis de tendencias
- **Modo `alertsOnly`** - Solo retornar alertas críticas/warnings
- **Recomendaciones automáticas** - Basadas en patrones detectados

### Nuevos parámetros
| Parámetro | Default | Descripción |
|-----------|---------|-------------|
| `days` | 7 | Ventana de días para analizar |
| `includeDashboard` | true | Incluir dashboard ASCII |
| `alertsOnly` | false | Solo retornar alertas |
