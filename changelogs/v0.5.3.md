# v0.5.3 - Globalizar análisis de acoplamiento + 3 estrategias educativas

## 🚀 Features

### split_large_file: Análisis de acoplamiento con 3 estrategias
- **Estrategia 1 - Fail Fast Educativo**: Cuando un archivo no puede dividirse, retorna reporte detallado con:
  - Porcentaje de acoplamiento (% átomos interconectados)
  - Mensaje explicativo del por qué no se puede dividir
  - Sugerencias de refactorización paso a paso
- **Estrategia 2 - Bottom-Up**: Identifica funciones hoja (no llaman a nadie)
  - Lista de funciones que pueden extraerse sin riesgo a `utils.js`
  - Prioridad HIGH para extracción segura
- **Estrategia 3 - Clustering**: Identifica nodos centrales (más llamados)
  - Top 5 funciones más conectadas
  - Sugiere refactorizarlas a módulo independiente

### detect_folderization_opportunities: Nuevo detector de acoplamiento alto
- **Nuevo parámetro**: `includeCoupling` (default: true)
- **Nuevo tipo de alerta**: `high_coupling`
- **Análisis por archivo monolítico** (>300 líneas):
  - Obtiene átomos del archivo
  - Enriquece con `call_graph` para dependencias
  - Ejecuta `analyzeCoupling` para métricas
  - Reporta si `canSplit = false` o acoplamiento > 70%
- **Campos en alerta**:
  - `couplingPercentage`: % de acoplamiento
  - `connectedAtoms` / `totalAtoms`: átomos conectados
  - `leafNodes`: funciones hoja para extracción
  - `centralNodes`: funciones centrales para refactorizar
  - `details.leafNodesCount`, `details.centralNodesCount`

### split_large_file: Dependencias desde call_graph
- **Enriquecimiento de átomos**: `analyzeFileForSplit` ahora consulta `call_graph` table
- **Campo `calls` poblado**: Cada átomo tiene sus llamadas internas
- **Agrupación dependiente**: `groupByExports` usa `calls` para incluir dependencias

## 🔧 Technical Improvements

### split-large-file-helpers.js
- **Nueva función `analyzeCoupling`**:
  - Calcula % de acoplamiento basado en dependencias
  - Identifica hojas (leaf nodes): funciones que no llaman a nadie
  - Identifica nodos centrales: funciones más llamadas (≥3 callers)
  - Retorna `canSplit`: si el archivo puede dividirse (< 80% acoplamiento)
- **Nueva función `generateSuggestions`**:
  - Genera sugerencias por estrategia (bottom-up, clustering, fail-fast)
  - Prioridades: critical, high, medium
  - Items con acciones concretas

### detect-folderization-opportunities.js
- **Nueva función `detectHighCoupling`**:
  - Analiza archivos monolíticos (>300L)
  - Enriquece átomos con call_graph
  - Ejecuta analyzeCoupling por archivo
  - Retorna alertas con métricas de acoplamiento
- **Nuevo parámetro `includeCoupling`**: Controla si se ejecuta análisis de acoplamiento
- **Summary actualizado**: Incluye `high_coupling` en `byType`
