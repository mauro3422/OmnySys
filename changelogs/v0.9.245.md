# v0.9.245

## Summary
- Aligned `CacheInvalidator` with the SQLite-backed `UnifiedCacheManager` API so invalidation now clears persisted cache state, not only the legacy in-memory fallback.
- Updated the cache invalidator tests to mock the public `invalidate()` surface instead of the retired `ramCache.delete()` path.
- Renamed internal RAM helper methods to reduce conceptual-duplicate noise in the watcher.
