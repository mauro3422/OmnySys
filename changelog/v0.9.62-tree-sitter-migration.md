# v0.9.62 - Complete Tree-sitter Migration & Stabilization

**Full Migration to Tree-sitter (Layer A)**: Transición completa de extractores basados en regex a análisis sintático preciso mediante Tree-sitter. Estabilización del servidor MCP tras la eliminación de código legacy redundante.

### Key Achievements

1. **High-Precision Analysis**: Los trackers de variables globales, singletons y closures ahora utilizan el scope real detectado por Tree-sitter.
2. **MCP Server Stability**: Eliminados 3 errores críticos de `ERR_MODULE_NOT_FOUND` causados por dependencias circulares y archivos legacy eliminados.
3. **Enhanced Metadata Exposure**: La herramienta `get_function_details` ahora expone `sharedStateAccess`, `eventEmitters`, `eventListeners` y el `scopeType` detectado por AST.
4. **Bridge Fix**: Corregido bug en el puente de integración que limitaba el análisis al código de la función; ahora procesa el contexto completo del archivo.
