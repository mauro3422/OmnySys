# v0.9.63 - Semantic Intelligence & Precise Mutations

**Active Semantic Analysis**: Introducción de validación activa de flujo de datos y detección de ambigüedades mediante AST. Estabilización de las mutaciones de código mediante focalización por símbolos.

### Key Achievements

1. **Precise Atomic Mutations**: `atomic_edit` ahora permite el uso de `symbolName`. Tree-sitter localiza el rango exacto de la función, eliminando fallos por discrepancias de formato o duplicidad de texto fuera del objetivo.
2. **Shadowing Detector**: Nueva herramienta en `detect_patterns` que identifica variables locales o parámetros que ocultan (shadow) variables de módulo o globales, previniendo errores lógicos sutiles.
3. **Invariant Checker (Secret Leakage)**: Implementación de validación de flujo de datos para detectar si variables sospechosas de contener secretos (tokens, passwords) llegan a sumideros de logging o consola.
4. **Standardized Internal Imports**: Refactorización de las rutas de importación en las nuevas herramientas de Layer A para usar alias de paquete (`#layer-a`, `#utils`), mejorando la robustez del hot-reload.
